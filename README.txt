prefadoros
==========
Πρόκειται για μια διαδικτυακή version του παιχνιδιού της πρέφας.
Το παιχνίδι βασίζεται σε βάση δεδομένων την οποία προσπελαύνουν παίκτες
μέσω του browser που χρησιμοποιούν.

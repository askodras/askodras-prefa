#
Για να εμφανίσετε τα φύλλα σας στο χώρο συζήτησης του τραπεζιού,
κάντε κλικ στο εικονίδιο \
<img src="images/trapoula/xromaS.png" class="mottoIkonidio" /> \
στο control panel. Στην παλέτα που θα
εμφανιστεί κάντε κλικ στο εικονίδιο \
<img src="images/preDianomi.png" class="mottoIkonidio" /> \
για τα φύλλα της προηγούμενης
διανομής, ή στο εικονίδιο \
<img src="images/nowDianomi.png" class="mottoIkonidio" /> \
για τα φύλλα της τρέχουσας διανομής.
#
Μπορείτε να αρχειοθετήσετε προσωπικές σας πληροφορίες που αφορούν
στους παίκτες του «Πρεφαδόρου», π.χ. τηλέφωνα, διευθύνσεις, σχόλια
κλπ, κάνοντας τις κατάλληλες προσθήκες στις πληροφορίες προφίλ των
παικτών αυτών. Οι προσθήκες αυτές είναι ορατές μόνο από εσάς.
#
Αν θέλετε να ακυρώσετε την τελευταία σας κίνηση, κάντε κλικ στο εικονίδιο \
<img src="images/controlPanel/bugFix.png" class="mottoIkonidio" /> \
του control
panel. Μπορείτε να ακυρώσετε περισσότερες από μία κινήσεις και είναι καλό να ακυρώσετε
κινήσεις μέχρι κάποιο ασφαλές σημείο, π.χ. μέχρι την προηγούμενη μπάζα. Όσο ακυρώνετε
κινήσεις, κανείς από τους παίκτες δεν μπορεί να κάνει οποιαδήποτε κίνηση μέχρι να κάνετε
κλικ στο εικονίδιο \
<img src="images/controlPanel/anamoniAkirosis.png" class="mottoIkonidio" />.
#
#
Για να αντικαταστήσετε κάποιον παίκτη που έχει αποχωρήσει
από κάποιο τραπέζι στο οποίο είστε θεατής, κάντε κλικ στο
εικονίδιο \
<img src="images/controlPanel/theatis.png" class="mottoIkonidio" /> \
του control panel.
#
Για να γίνετε από παίκτης θεατής σε κάποιο τραπέζι, κάντε κλικ
στο εικονίδιο \
<img src="images/controlPanel/theatis.png" class="mottoIkonidio" /> \
του control panel.
#
Το <em>control panel</em> είναι η κεντρική στήλη εργαλείων του «Πρεφαδόρου».
Τα βολικότερα εργαλεία σε κάθε φάση του παιχνιδιού είναι αυτά στα οποία
εμφανίζεται το <img src="images/controlPanel/kafedaki.png" class="mottoIkonidio" />, \
αλλά μπορείτε να δείτε και άλλες σειρές εργαλείων
κάνοντας κλικ στο εικονίδιο \
<img src="images/controlPanel/4Balls.png" class="mottoIkonidio" /> \
στην κορυφή του control panel.
#
Το εικονίδιο <img src="images/controlPanel/default.png" class="mottoIkonidio" /> \
επαναφέρει τη βασική σειρά εργαλείων στο control panel.
Η βασική σειρά εργαλείων περιέχει το \
<img src="images/controlPanel/kafedaki.png" class="mottoIkonidio" /> \
και παρέχει τα βολικότερα εργαλεία
σε κάθε φάση του παιχνιδιού.
#
Για να εισέλθετε σε κάποιο τραπέζι στο οποίο έχετε προσκληθεί, αρκεί να
κάνετε κλικ στην πρόσκληση, αποφεύγοντας όμως τα εικονίδια \
<img src="images/Xgreen.png" class="mottoIkonidio" /> και \
<img src="images/Xred.png" class="mottoIkonidio" />.
#
Οι παίκτες που έχετε χαρακτηρίσει ως φίλους, μπορούν να σας
προσκαλέσουν σε τραπέζι ακόμη και αν είστε απασχολημένος.
#
Αν είστε απασχολημένος, αλλά ενδιαφέρεστε να παίξετε σε κάποιο
τραπέζι, μπείτε θεατής στο τραπέζι. Έτσι οι παίκτες του τραπεζιού
μπορούν να σας προσκαλέσουν εμμέσως μέσω της πρόσκλησης ομιλίας.
#
Δικαίωμα συμμετοχής στη συζήτηση που εξελλίσσεται σε κάποιο τραπέζι
έχουν όσοι παίκτες και θεατές έχουν πρόσκληση στο τραπέζι αυτό.
#
Το κιτάπι παρέχεται μόνο για συναισθηματικούς λόγους. Τα κέρδη
και οι ζημίες των παικτών μετά το μοίρασμα της υπόλοιπης κάσας
εμφανίζονται στο τραπέζι ανά πάσα στιγμή, επομένως δεν υπάρχει
απολύτως κανένας πρακτικός λόγος να έχετε ανοικτό το κιτάπι.
#
Αν εξέλθετε από κάποια παρτίδα η οποία δεν έχει τελειώσει, τότε
στη θέση σας μπορεί να εισέλεθει οποιοσδήποτε διαθέτει πρόσκληση
στο τραπέζι.
#
Οι παρτίδες αρχειοθετούνται αμέσως μετά την έξοδο και των τριών
παικτών. Αν ένας ή δύο παίκτες δεν εξέλθουν από το τραπέζι, τότε
το τραπέζι αρχειοθετείται αρκετή ώρα μετά την τελευταία επικοινωνία
των παικτών αυτών με τον «Πρεφαδόρο».
#
Μπορείτε να «ανεβάσετε» φωτογραφία προφίλ κάνοντας κλικ στο login
σας, όπως αυτό εμφανίζεται στο επάνω δεξιά μέρος της οθόνης τού
φυλλομετρητή σας.
#
Για να διορθώσετε τα στοιχεία του λογαριασμού σας, μπορείτε να
κάνετε κλικ στο login σας όπως αυτό εμφανίζεται στο επάνω δεξιά
μέρος της οθόνης τού φυλλομετρητή σας.
#
Δεν είναι απαραίτητο να διαγράψετε τις προσκλήσεις που σας έχουν
αποσταλεί. Οι προσκλήσεις που σας έχουν αποστείλει οι παίκτες ενός
τραπεζιού διαγράφονται αυτόματα με την αρχειοθέτηση της παρτίδας.
#
Αν θέλετε να στείλετε μαζικά προσκλήσεις σε όλους τους θεατές
του τραπεζιού στο οποίο συμμετέχετε ως παίκτης, μπορείτε να
κάντε κλικ στο εικονίδιο \
<img src="images/controlPanel/theatis.png" class="mottoIkonidio" /> \
που εμφανίζεται πάνω από την περιοχή
των ποσκλήσεων.
#
Για να διαγράψετε όλες τις προσκλήσεις που σας αφορούν, κάντε κλικ
στο εικονίδιο \
<img src="images/Xred.png" class="mottoIkonidio" /> \
που εμφανίζεται πάνω από την περιοχή των προσκλήσεων.
#

Στο παρόν directory δημιουργούνται διάφορα lock files/directories.

neoTrapezi
----------
Δεν επιτρέπει τη δημιουργία τραπεζιού και προβάλλει το μήνυμα που περιέχει.
Αν το αρχείο είναι κενό, τότε προβάλλεται γενικό μήνυμα.
Αν περιέχεται κάποιο μήνυμα, τότε το μήνυμα αυτό μπορεί να είναι HTML, αλλά
θα προβληθεί στο FYI message line, επομένως θα είναι μια γραμμή και μάλιστα
όχι πολύ μεγάλου μήκους.

klista.php
----------
Δεν επιτρέπει καμία κίνηση στο site. Προβάλλεται το (HTML) περιεχόμενο,
εφόσον υπάρχει, αλλιώς προβάλλεται γενικό μήνυμα.
